# AI Portfolio

A Django portfolio skeleton with pages for home, about, projects, project details, skills, resume, and contact.

## Run Locally

```powershell
.\.venv\Scripts\python.exe manage.py runserver
```

Then open http://127.0.0.1:8000/.

## Pages

- Home: `/`
- About: `/about/`
- Projects: `/projects/`
- Project detail: `/projects/sample-project/`
- Skills: `/skills/`
- Resume: `/resume/`
- Contact: `/contact/`
